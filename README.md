# Project Overview

This repository contains two main parts:

1.  **Travel Time Estimation**

    Processes NYC TLC taxi trip data and estimates travel time
    statistics for the network.

2.  **Load Estimation / Stochastic Routing Optimization**

    Uses the processed network and trip queries to run reliability-aware
    routing and evaluate system load.

Each part has its own directory and detailed README. Please check the
README inside each folder for setup, input format, parameters, and
running instructions.

The PDF file attached is our project report, which may help you have a general understanding of our work.
